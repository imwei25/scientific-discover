科研助手 Windows 桌面安装包。

## v0.2.2 修复/改进
- **安装不再报"写不了 sidecar.exe"**：安装程序在覆盖前会自动结束仍在运行的后台 `sidecar.exe`，升级更顺。
- **侧栏底部「⚙ 设置」按钮在矮屏/缩放下也能看见**：导航区改为可滚动，底部状态+设置始终固定可见（此前会被裁掉，导致找不到设置按钮）。
- **首页自适应屏幕**：hero 不再硬撑高度、窄窗口下两栏堆叠为单栏，首页不再显示不全。

## v0.2.1 修复/改进
- **API key 设置入口常驻**：侧栏底部新增「⚙ 设置」按钮，随时可打开配置向导改 key/模型；未配置时的提示改为「⚠ 未配置密钥 · 点此设置」可直接点开（不再让用户去找 backend/.env）。
- **项目选择器移回主区右上角**。
- 修正「开始文献调研 / 提取 PICO」等操作按钮高度不对齐。

## v0.2.0 新增
- **多项目管理**：一篇论文一个项目，侧栏切换、后端落盘、各模块按项目隔离保存。
- **应用内配置向导**：首次打开引导填 API key，可在页面里直接编辑/切换模型并**在线测试 key 是否可用**，不必手动改文件。
- **帮助卡片**：9 个子工具旁的“?”说明何时用、怎么用。
- **病例数据脱敏**：上传含 PHI 的数据时自动检测并脱敏。
- **参考文献导入/导出**：支持 .ris / .bib / .enw。
- 数据分析新增森林图 / KM 生存曲线 / ROC；UI 升级（Toast 提示、Diff 视图、Ctrl+K 命令面板、字体调节、Lucide 图标）。

## 安装与配置
- 下载 `*-setup.exe` 双击安装，终端用户无需安装 Python 或任何依赖。
- 首次打开按向导填入 `LLM_API_KEY`（默认 DeepSeek，也可换硅基流动/OpenAI/Claude）即可；
  配置存于 `%APPDATA%\科研助手\.env`，重装不丢。
- 数据分析、画图、脱敏均在本机完成，数据不出网。
