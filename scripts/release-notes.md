## v0.2.7 修复/改进
- **修复：配置硅基流动 / OpenAI 时"测试连接不通、也保存不了"**。此前测试连接没按所选供应商解析接口地址，会把你的 key 误发到 DeepSeek 的接口 → 一律提示 key 无效；而"保存"要先测通才可点，于是连带存不了。现在测试连接会按供应商用正确的接口地址（与保存一致）。

## 安装与配置
- 下载 `*-setup.exe` 双击安装，终端用户无需安装 Python 或任何依赖。
- 首次打开按向导填入 `LLM_API_KEY`（默认 DeepSeek，也可换硅基流动/OpenAI/Claude）即可；
  配置存于 `%APPDATA%\科研助手\.env`，重装不丢。
- 数据分析、画图、脱敏均在本机完成，数据不出网。
